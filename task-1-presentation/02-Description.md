# Task 1 – Digital Literacy Awareness Infographic

## Tool Used
**Canva** (canva.com) – free online design platform

## About the Infographic
This one-page infographic was created to help batchmates understand Digital Literacy and why it matters in today's world. 

## Topics Covered
1. **What is Digital Literacy?** – The ability to use digital technologies effectively, safely, and responsibly.
2. **Useful Digital Tools for Students** – Google Workspace, Canva, GitHub, Notion, Grammarly
3. **Safe Internet Practices** – Using strong passwords, avoiding phishing links, enabling 2FA
4. **Professional Online Presence** – LinkedIn, GitHub profile, ResearchGate

## Design Notes
The infographic uses a clean blue-and-white colour scheme with icons and short text to make it easy to scan and understand.
